# Enterprise Governance Dashboard

## Project Overview

This project is an enterprise-level Power BI Governance Dashboard designed to monitor data quality, validation issues, duplicate records, metadata completeness, and governance KPIs across customer, product, employee, and sales datasets.

The solution provides interactive analytics, drillthrough reporting, tooltip insights, and governance monitoring to support business decision-making and improve data reliability.

---

## Features

- Executive KPI Dashboard
- Customer Data Governance Dashboard
- Product Metadata Governance Dashboard
- Employee Governance Monitoring
- Sales Validation Dashboard
- Drillthrough Analysis
- Tooltip Reporting
- Interactive Page Navigation
- Business Insights Reporting

---

## Tools & Technologies

- Power BI
- SQL
- Python
- Excel
- Data Validation
- Data Governance

---

## Key KPIs

- Governance Score
- Duplicate %
- Missing %
- Metadata Completeness
- Invalid Records
- Active Customers
- Total Transactions
